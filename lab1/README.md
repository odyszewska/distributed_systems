## `zad1`
Contains files for **Task 1**:
- `JavaUdpServer.java` – UDP server in Java
- `JavaUdpClient.java` – UDP client in Java

This task implements two-way UDP communication between **Java client** and **Java server**.

## `zad2`
Contains files for **Task 2**:
- `JavaUdpServer.java` – UDP server in Java
- `PythonUdpClient.py` – UDP client in Python

This task implements UDP communication between **Java** and **Python** and sends the text message: `żółta gęś`.

## `zad3`
Contains files for **Task 3**:
- `JavaUdpServer.java` – UDP server in Java
- `PythonUdpClient.py` – UDP client in Python

This task implements sending a numeric value between **Python client** and **Java server** using **little-endian byte order**.  
The Python client sends number `300`, the Java server reads it, increments it by `1`, and sends it back.
