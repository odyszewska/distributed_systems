# Distributed Systems – AGH University of Krakow

This repository contains small projects prepared for the **Distributed Systems** course at **AGH University of Krakow**.

The repository is organized into:
- **7 homework assignments**
- **7 laboratory exercises**

Each task or lab is stored in a separate folder.  
Every folder contains its own `README.md` file with a description of the project, requirements, and instructions on how to run it.
